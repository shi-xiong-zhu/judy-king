for循环

for (语句 1; 语句 2; 语句 3) {
     要执行的代码块
}
  1. 语句 1 在循环（代码块）开始之前执行。
  2. 语句 2 定义运行循环（代码块）的条件。
  3. 语句 3 会在循环（代码块）每次被执行后执行。
  • 要执行的代码块可以和语句3换书写位置

# 显式原型, 隐式原型，原型链

  • 定义的函数对象时产生一个显式原型对象里面没有我们自己的属性（Foo.prototype={constructor:Foo}）

  • Fooe手动指向的Foo.prototype没有constructor属性需要时手动添加

  • new一个实例对象产生一个隐式原型对象里面没有我们自己的属性（this._proto_=this.prototype）

  • 所有函数（对象）｛Ｆｏｏ，Ｏｂｊｅｃｔ｝都是由Function new出来的包括Function自己 ,(function(){}).constructor = Function所以所有函数（对象）都有一个显式原型和一个隐式原型

  • Ｏｂｊｅｃｔ原型对象的隐式原型属性为ｎｕｌｌ（隐式原型链尽头）
